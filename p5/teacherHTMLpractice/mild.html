Before we get to fun stuff we need to make sure we all understand the basics of HTML!

HTML is a programming mark up language that gives structure to webpages.

We can think of it like a page's bones.

You can do so much with HTML:

  • Define paragraphs
  • Create divs (boxes) to hold and organize content
  • Write lists (ordered or not!)
  • You can even change text color,but it's often better to do this in a separate CSS style sheet.

HTML uses tags (opening and closing) and a hierarchy system to structure webpages. You can even give seperate elements classes and IDs to make groupings easier!
